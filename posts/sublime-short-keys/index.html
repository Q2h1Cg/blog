快捷键 功能
cmd + p 当前项目中快速搜索文件
cmd + / 行内注释
cmd + shift + / 块注释
cmd + shift + f 指定目录下所有文件中搜索内容
cmd + r 快速跳转到文件中的某个函数处
ctrl + g 跳转到指定行
cmd + d 选中相同字符
ctrl + shift + m 选中区间内容
